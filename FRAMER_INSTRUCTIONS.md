## üö® TROUBLESHOOTING

### Problem: Can't find Feature (copy) template
**Solution:**
- Check "Projects" list in Framer dashboard
- It might be called "Feature Template" or just "Feature"
- If missing, re-copy from marketplace

### Problem: Can't edit text
**Solution:**
- Make sure you're in "Edit" mode (not Preview)
- Double-click on text to enter edit mode
- Some text might be in a component - click "Unlink" to edit

### Problem: Pricing cards won't update
**Solution:**
- Pricing cards are often components
- Right-click ‚Üí "Unlink from Master" to edit freely
- Or update the component properties in right panel

### Problem: Mobile looks broken
**Solution:**
- Framer templates are usually responsive by default
- Check "Responsive" tab in right panel
- Adjust breakpoints if needed
